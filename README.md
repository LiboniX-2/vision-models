# Vision Models Lab

A collection of classic and modern computer vision architectures implemented using PyTorch. This repository focuses on understanding deep learning models by rebuilding them from scratch and experimenting with transfer learning.

## Models Implemented

- LeNet
- AlexNet
- VGG16
- VGG19
- ResNet-18
- GoogLeNet
- Inception v2
- Inception v3
- U-Net (coming soon)

## Project Goals

- Understand how major CNN architectures are built internally
- Implement models from scratch using PyTorch
- Compare performance across different architectures
- Experiment with transfer learning and fine-tuning
- Build strong intuition for computer vision models

## Tech Stack

- Python
- PyTorch
- NumPy
- Matplotlib
- Jupyter Notebook (optional)

## Key Features

- From-scratch implementations of CNN architectures
- Transfer learning experiments on standard datasets
- Training and evaluation pipelines
- Visualization of training loss and accuracy

## Notes

- Some models are implemented fully from scratch, while others use transfer learning with pretrained weights.
- Large datasets and trained weights are not included in this repository.
